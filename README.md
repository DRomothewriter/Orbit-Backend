# Orbit
Orbit será una aplicación backend desarrollada en TypeScript que implementará un sistema de mensajería en tiempo real con funcionalidades extras que la distinguen de las demás como crear ToDo lists por grupo para que se puedan organizar. Su objetivo es permitir la comunicación instantánea entre usuarios mediante el uso de WebSockets y un servidor construido con Express y Socket.IO.

# .env
**PORT** = puerto en el que va a correr.

**MONGO_URI** = tu mongo URI.
